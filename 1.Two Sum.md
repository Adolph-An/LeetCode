# 题目：输出给定数组中和为目标值的下标，多个答案只需输出一个
## LeetCode难度：简单
```c
int* twoSum(int* nums, int numsSize, int target) {
    int *result=(int *)malloc(sizeof(int)*2);
    for(int i=0;i<numsSize-1;i++){
        for(int j=i+1;j<numsSize;j++){
            if((nums[i]+nums[j])==target){
                result[0]=i;
                result[1]=j;
                return result;
            }
        }
    }
    return NULL;
}
```
暴力解题，两层循环
